# SoundCloud-TUI Player


Installation:

Linux

```
git clone https://github.com/laguser/soundcloud-tui
cd soundcloud-tui
python -m venv venv
venv\Scripts\activate
pip install textual pygame-ce yt-dlp requests
python install.py
```
(Обязательно! - Сделайте все как на туториале!!!)
---
```
#Arch
sudo pacman -S python
#Ubuntu
sudo apt install python
```
